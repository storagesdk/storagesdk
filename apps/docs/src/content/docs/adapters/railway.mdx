## Overview

A branded alias of the [Tigris adapter](/adapters/tigris) because [Railway Buckets](https://docs.railway.com/storage-buckets) run on Tigris. Behavior, snapshot/fork support, and `storage.raw` are identical to the Tigris adapter — the only difference is the import name, which matches the platform you're working in.

## Configuration

```ts
import { Storage } from '@storagesdk/core';
import { railway } from '@storagesdk/adapters/railway';

const storage = new Storage({
  adapter: railway({
    bucket: 'agent-runs',
    accessKeyId: process.env.ACCESS_KEY_ID,
    secretAccessKey: process.env.SECRET_ACCESS_KEY,
  }),
});
```

```ts
railway({
  bucket: string;
  accessKeyId: string;
  secretAccessKey: string;
  endpoint?: string;
  forcePathStyle?: boolean;
});
```

## Notes

- Peer dependency: `@tigrisdata/storage`.
- Behavior and semantics match `@storagesdk/adapters/tigris` exactly — see the [Tigris page](/adapters/tigris) for full details.

## Compatibility

Identical to [Tigris](/adapters/tigris#compatibility) — same SDK underneath.
